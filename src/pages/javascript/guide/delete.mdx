# Delete

You can easity delete items that match your `filter` criteria using OneNode API `DELETE` request.

## Delete items

```javascript copy
const filter = { id: "id_of_the items" };

const queryParams = new URLSearchParams({
  filter: filter,
}).toString();

const response = await fetch(
  `https://api.onenode.ai/v1/resource?${queryParams}`,
  {
    method: "DELETE",
    headers: {
      "Content-Type": "application/json",
      Authorization: `Bearer ${ONENODE_API_KEY}`,
    },
  }
);

console.log(response);

/*
Response {
  body: ReadableStream,
  bodyUsed: false,
  headers: Headers,
  ok: true,
  redirected: false,
  status: 204,
  statusText: "No Content",
  type: "cors",
  url: "https://api.onenode.ai/v1/resource?filter=%7B%22id%22%3A%22id_of_the_items%22%7D"
}
*/
```

Now you've successfully deleted items using OneNode API.
