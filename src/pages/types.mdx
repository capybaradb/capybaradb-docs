# Supported Data Types

At the time being, Semantic Cloud currently supports a subset of data types listed below. By the end of 2024, OneNode plan to adopt all major data types, not limited to text and image.
For `type` specification in `_embed_targets`, use MIME type displayed below. You can include multiple data types in one item.

| Data Type  | Extension(s) | MIME Type              |
| ---------- | ------------ | ---------------------- |
| Plain Text | .txt         | text/plain             |
| JPEG       | .jpg, .jpeg  | image/jpeg             |
| PNG        | .png         | image/png              |
| GIF        | .gif         | image/gif              |
| BMP        | .bmp         | image/bmp              |
| TIFF       | .tiff, .tif  | image/tiff             |
| WEBP       | .webp        | image/webp             |
| SVG        | .svg         | image/svg+xml          |
| HEIF       | .heif, .heic | image/heif, image/heic |
